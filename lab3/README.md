# Третя лабораторна робота

## Опис завдання
Потрібно було створити Jupyter Notebook зі скриптами для обробки та візуалізації даних, пов’язаних із серцевими захворюваннями.

Основні етапи роботи:
- Завантаження даних
  - Автоматично завантажити файли з даними про серцеві захворювання.
  - Реалізувати перевірку, щоб не завантажувати однакові файли повторно.
  - На сервері знаходиться zip-архів з датасетом Heart Disease.
  - Розпакувати архів та зчитати дані з 4 файлів:
    - `processed.cleveland.data`
    - `processed.hungarian.data`
    - `processed.switzerland.data`
    - `processed.va.data`
  - Об’єднати файли в один CSV-файл `heart_disease.csv`.
  - Додати стовпець `territory`, який вказує джерело даних.

- Data reading and cleaning
  - Зчитати об’єднаний CSV-файл у pandas DataFrame.
  - Видалити зайві рядки і текст.
  - Привести стовпці до коректних типів даних.
  - Замінити закодовані категоріальні значення на зрозумілі текстові позначення.
  - Створити стовпець `disease` для позначення наявності або відсутності захворювання.

- Аналіз даних та візуалізація
  - Побудувати графік залежності максимальної ЧСС від віку (`scatterplot`).
  - Побудувати графік середнього рівня холестерину залежно від віку (`lineplot`).
  - Побудувати гістограму розподілу пацієнтів за віком у 5 заданих діапазонах.
  - Побудувати `heatmap` кореляції числових ознак.
  - Побудувати `violinplot` значень `oldpeak` за діагнозом.
  - Побудувати `boxplot` розподілу максимальної ЧСС за наявністю хвороби.
  - Побудувати `barplot` середнього значення `oldpeak` за діагнозом.
  - Побудувати `pairlot` для визначення зміни показників за діагнозом.


## Інструкція із запуску

### Підготовчий єтап
Завантажте віртуальне середовище в вашому проєкті
```
python -m venv .venv
```

Активуйте venv

Windows
```
.\.venv\Scripts\activate
```
Linux\MacOS
```
source .venv/bin/activate
```

Встановіть Jupyter Notebook для роботи з .ipynb
```
pip install jupyter
```

Встановити всі залежності з файлу requirements.txt
```
pip install -r requirements.txt
```

### Робота з даними

Для початку роботи з даними у `medical.ipynb` потрібно виконати процедури:
- Ініціалізації бібліотек.
- `download_file()` - Для завантаження архіву з серверу.
- `csv_to_df()` - Для перетворення файлу .csv у pandas DataFrame.
- Для виведень графіків:
  - `scatterplot()` - Побудова графіку `scatterplot`.
  - `lineplot()` - Побудова графіку `lineplot`.
  - `hist_by_bins()` - Побудова гістограми з вказаними діапазонами.
  - `heatmap()` - Побудова графіку `heatmap`.
  - `violinplot()` - Побудова графіку `violinplot`.
  - `boxplot()` - Побудова графіку `boxplot`.
  - `pairplot()` - Побудова графіку `pairplot`.

## Тлумачення показників

- `age` - Вік пацієнту.
- `sex` - Стать пацієнту.
- `trestbps` - Артеріальний тиск у стані спокою.
- `chol` - Рівень холестирину в крові.
- `thalach` - Максимальна досягнута частота серцевих скорочень.
- `oldpeak` - Зниження сегмента ST відносно стану спокою під час фізичного навантаження.
- `territory` - Территорія джерела даних.
- `disease` - Наявність або відсутність серцевого захворювання.
