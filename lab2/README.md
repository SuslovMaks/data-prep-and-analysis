# Друга лабораторна робота

Вона розділена на дві частини:
1. **Індекс здоров’я рослин (VHI)** для українських областей.
2. **Дані по споживанню електроенергії у будинках** для детального аналізу.

## Опис завдання
### Перша частина
Потрібно було створити Jupyter Notebook зі скриптами для обробки даних пов'язаними з Індексом здоров'я рослин (VHI).

Основні этапи роботи:
  - Завантаження даних
    - Автоматично завантажити текстові файли з індексом VHI для кожної області України.
    - При збереженні файлів додати дату і час завантаження.
    - Реалізувати перевірку, щоб не завантажувати однакові файли повторно.
  - Data reading and cleaning
    - Завантажені файли зчитати у pandas DataFrame.
    - Видалити зайві рядки і текст.
    - Прибрати непотрібні стовпці.
    - Додати стовпці з назвою області та її індексом.
  - Процедура зміни індексів
    - Дані NOAA нумерують області за англійською абеткою, потрібно перерахувати їх за українською абеткою.
  - Аналіз даних (Створення процедур для)
    - Отримання ряду VHI для області за певний рік.
    - Отримання ряду VHI за діапазон років для кількох областей.
    - Пошуку мінімуму, максимуму, середнього та медіани.

### Друга частина
Потрібно було створити Jupyter Notebook зі скриптами для обробки даних пов'язаними з даними по споживанню електроенергії у будинках.

Основні этапи роботи:
  - Завантаження даних
    - Автоматично завантажити текстовий файл з даними по споживанню електроенергії у будинках.
    - На сервері знаходиться zip-архів, використовуємо бібліотеку zipfile для розпакування.
    - Реалізувати перевірку, щоб не завантажувати однакові файли повторно.
  - Data reading and cleaning
    - Завантажені файли зчитати у pandas DataFrame.
    - Видалити зайві рядки і текст.
    - Прибрати непотрібні стовпці.
    - Змінити тип данних у стовпцях.
    - Додати стовпець з повною датою.
  - Аналіз даних (Створення процедур для)
    - Отримання записів, у яких активна споживна потужність перевищує 5 кВт.
    - Отримання записів, у яких сила струму лежить в межах 19-20 А, для них виявити ті, у яких пральна машина та холодильних споживають більше, ніж бойлер та кондиціонер.
    - Отримання записів, де випадковим чином обрано 500000 записів, для них обчислити середні величини усіх 3-х груп споживання електричної енергії.
    - Отримання записів, у яких вечірня споживна потужність перевидує 6 кВт, серед них визначити тих, у яких основне споживання електроенергії у вказаний проміжок часу припадає на пральну машину, сушарку, холодильник та освітлення, а потім обрати кожен третій результат із першої половини та кожен четвертий результат із другої половини.
    - Нормалізувати дані.
    - Стандартизувати дані.
    - Підрахувати коефіцієнт Пірсона та Спірмена.
    - Провести One Hot Encoding категоріального атрибута.
  - Профіліювання часу виконання

### Перша частина
Для початку роботи з даними у `vhi_notebook.ipynb` потрібно виконати процедури:
- Ініціалізації бібліотек.
- `download_file()` - Для завантаження даних з сайту NOAA.
- `csv_to_df()` - Для перетворення файлу .csv у pandas DataFrame.
- `change_indexes()` - Для змінення індексів з англійської абетки на українську.
- Вибірки:
  - `filter_province_year(df, province, year)` - Ряд VHI для області за вказаний рік.
    - `df` - DataFrame з якого потрібно зробити вибірку.
    - `province` - Індекс необхідної області.
    - `year` - Рік вибірки.
  - `filter_provinces_years(df, provinces, years)` - Ряд VHI за вказаний діапазон років для вказаних областей.
    - `df` - DataFrame з якого потрібно зробити вибірку.
    - `provinces` - Індекси необхідних областей.
    - `years` - Роки вибірки.
  - `filter_stats_provinces_years(df, provinces, years)` - Пошук екстремумів (min та max) для вказаних областей та років, середнього, медіани.
    - `df` - DataFrame з якого потрібно зробити вибірку.
    - `provinces` - Індекси необхідних областей.
    - `years` - Роки вибірки.

### Друга частина
Для початку роботи з даними у `power_notebook.ipynb` потрібно виконати процедури:
- Ініціалізації бібліотек.
- `download_file()` - Для завантаження даних з серверу.
- `txt_to_df()` - Для перетворення файлу .txt у pandas DataFrame.
- Вибірки:
  - `filter_power_over_5kw(df)` - Записи, у яких активна споживна потужність перевищує 5 кВт.
  - `filter_between_19_20(df)` - Записи, у яких сила струму лежить в межах 19-20 А, для них виявити ті, у яких пральна машина та холодильних споживають більше, ніж бойлер та кондиціонер.
  - `filter_random_mean(df)` - Записи, де випадковим чином обрано 500000 записів, для них обчислити середні величини усіх 3-х груп споживання електричної енергії.
  - `filter_evening(df)` - Записи, у яких вечірня споживна потужність перевидує 6 кВт, серед них визначити тих, у яких основне споживання електроенергії у вказаний проміжок часу припадає на пральну машину, сушарку, холодильник та освітлення, а потім обрати кожен третій результат із першої половини та кожен четвертий результат із другої половини.
- `normalization(df)` - Для нормалізації датасету.
- `standardization(df)` - Для стандартизації датасету.
- `pearson_correlation(df)` - Для підрахування коефіцієнту Пірсона.
- `spearman_correlation(df)` - Для підрахування коефіцієнту Спірмена.
- `one_hot_encoding(df)` - Для проведення One Hot Encoding'у.
- `time_profiling(df, times)` - Для профіліювання часу.
  - `df` - DataFrame з якого потрібно зробити вибірку.
  - `times` - Кількість виконаннь операцій.
