# Четверта лабораторна робота

## Опис завдання
Потрібно було створити програму з графічним інтерфейсом для побудови та аналізу гармонічного сигналу з накладеним шумом.

Основні етапи роботи:
- Побудова гармоніки
  - Реалізувати функцію `harmonic_with_noise()` для генерації гармоніки виду `y(t) = A * sin(ωt + φ)`.
  - Додати можливість накладання шуму із заданими параметрами:
    - `noise_mean` - середнє значення шуму
    - `noise_covariance` - дисперсія шуму
  - Через параметр `show_noise` реалізувати можливість показу або приховування шуму на графіку.

- Побудова графічного інтерфейсу
  - Створити головне вікно програми на `PyQt6`.
  - Вбудувати область для побудови графіків за допомогою `matplotlib`.
  - Додати слайдери для керування параметрами гармоніки:
    - амплітудою
    - частотою
    - фазовим зсувом
  - Додати слайдери для керування параметрами шуму:
    - середнім значенням
    - дисперсією
  - Додати слайдери для параметрів фільтра:
    - частота зрізу
    - порядок фільтра
  - Додати чекбокси:
    - `Показувати шум`
    - `Увімкнути фільтр`
  - Додати кнопки:
    - `Reset`
    - `Згенерувати новий шум`

- Логіка оновлення сигналу
  - Після зміни будь-якого параметра графік має одразу оновлюватись.
  - Якщо змінюються лише параметри гармоніки, шум не повинен генеруватись наново.
  - Якщо змінюються лише параметри шуму, гармоніка має залишатись незмінною.
  - Після натискання `Reset` мають відновлюватись початкові параметри та початкова реалізація шуму.

- Фільтрація сигналу
  - Реалізувати low-pass фільтрацію зашумленої гармоніки за допомогою `scipy.signal.iirfilter`.
  - Використати IIR-фільтр для наближення зашумленого сигналу до чистої гармоніки.
  - Відобразити відфільтрований сигнал поруч із початковим для порівняння.

- Візуалізація результату
  - На першому графіку відображати початковий сигнал:
    - або чисту гармоніку
    - або гармоніку з шумом
  - На другому графіку відображати результат фільтрації.
  - Для обох графіків додати підписи осей, заголовки, сітку та легенду.

## Робота з програмою

Після запуску програми відкривається вікно з двома графіками:

- `Початковий сигнал` - чиста або зашумлена гармоніка
- `Відфільтрований сигнал` - результат застосування IIR low-pass фільтра

У програмі доступні такі елементи керування:

- Параметри гармоніки:
  - Amplitude - амплітуда гармоніки
  - Frequency - частота гармоніки
  - Phase - фазовий зсув гармоніки
- Параметри шуму:
  - Noise Mean - середнє значення шуму
  - Noise Covariance - дисперсія шуму
  - Показувати шум - чекбокс для вмикання або вимикання відображення шуму
  - Згенерувати новий шум - кнопка для створення нової випадкової реалізації шуму
- Параметри фільтра:
  - Cutoff Frequency - частота зрізу low-pass фільтра
  - Filter Order - порядок IIR-фільтра
  - Увімкнути фільтр - чекбокс для вмикання або вимикання фільтрації
- Додатково:
  - Reset - відновлення початкових параметрів сигналу, шуму, фільтра та початкового шуму
## Основні функції програми:
- `harmonic_with_noise()` - генерація чистої гармоніки, шуму та зашумленого сигналу
- `apply_iir_lowpass()` - low-pass фільтрація сигналу через scipy.signal.iirfilter
- `update_plot()` - оновлення графіків після зміни параметрів
- `generate_new_noise()` - генерація нового шуму
- `reset_values()` - скидання всіх параметрів до початкових значень
