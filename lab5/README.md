# П'ята лабораторна робота
## Опис завдання

Потрібно було створити веб-додаток з використанням Streamlit для аналізу часових рядів VCI, TCI та VHI для областей України.

Основні етапи роботи:

- Завантаження даних
  - Автоматично завантажити CSV-файли з даними VCI, TCI та VHI для кожної області України з сайту NOAA.
  - При збереженні файлів додати дату і час завантаження.
  - Реалізувати перевірку, щоб не завантажувати однакові файли повторно.

- Data reading and cleaning
  - Завантажені CSV-файли зчитати у pandas DataFrame.
  - Видалити зайві стовпці.
  - Замінити некоректні значення `-1` на `NaN`.
  - Видалити рядки з порожніми значеннями.
  - Додати стовпець з індексом області.
  - Додати стовпець з назвою області.

- Створення веб-додатку
  - Створити dropdown список для вибору часового ряду:
    - `VCI`
    - `TCI`
    - `VHI`
  - Створити dropdown список для вибору області.
  - Створити slider для вибору інтервалу тижнів.
  - Створити slider для вибору інтервалу років.
  - Створити button для скидання всіх фільтрів до початкового стану.
  - Розмістити інтерактивні елементи в одній колонці, а таблицю та графіки — в іншій.

- Фільтрація даних
  - Фільтрувати дані за вибраною областю.
  - Фільтрувати дані за вибраним інтервалом тижнів.
  - Фільтрувати дані за вибраним інтервалом років.
  - Виводити тільки обраний часовий ряд `VCI`, `TCI` або `VHI`.

- Сортування даних
  - Створити checkbox для сортування за зростанням значень обраного показника.
  - Створити checkbox для сортування за спаданням значень обраного показника.
  - Реалізувати реакцію програми, якщо увімкнені обидва checkbox:
    - показати попередження;
    - повернути стандартне сортування за роком і тижнем.

- Відображення результатів
  - Створити три вкладки:
    - таблиця з відфільтрованими даними;
    - графік часового ряду для обраної області;
    - графік порівняння областей.
  - Перший графік відображає значення обраного показника для вибраної області за вказаний діапазон років та тижнів.
  - Другий графік відображає порівняння середніх значень обраного показника між усіма областями.
  - Обрана область на графіку порівняння виділяється окремим кольором.

## Інструкція із запуску
### Підготовчий єтап
Завантажте віртуальне середовище в вашому проєкті
```
python -m venv venv
```

Активуйте venv

Windows
```
.\venv\Scripts\activate
```
Linux\MacOS
```
source venv/bin/activate
```
Встановити всі залежності з файлу requirements.txt
```
pip install -r requirements.txt
```

### Запуск веб-додатку

Для запуску Streamlit-додатку виконайте команду:
```
streamlit run app.py
```
Після запуску у браузері відкриється веб-додаток для аналізу часових рядів VCI, TCI та VHI.

## Основні процедури

Для початку роботи з даними у `app.py` використовуються такі процедури:

- `file_already_exists(province_id)` - Перевіряє, чи вже існує файл з даними для певної області.
  - `province_id` - Індекс області.

- `download_file()` - Завантажує CSV-файли з сайту NOAA для всіх областей України.
  - Дані зберігаються у папку `data`.
  - У назву файлу додається дата і час завантаження.
  - Якщо файл для області вже існує, повторне завантаження не виконується.

- `csv_to_df()` - Зчитує всі CSV-файли з папки `data` і об’єднує їх в один pandas DataFrame.
  - Видаляє непотрібні стовпці `SMN` та `SMT`.
  - Замінює значення `-1` на `NaN`.
  - Видаляє порожні рядки.
  - Додає `province_id`.
  - Додає `province_name`.

- `set_default()` - Скидає всі інтерактивні елементи до початкового стану.
  - Обраний індекс повертається до `VCI`.
  - Обрана область повертається до першої області зі списку.
  - Інтервал тижнів повертається до повного діапазону.
  - Інтервал років повертається до повного діапазону.
  - Checkbox сортування скидаються у значення `False`.

- `apply_dark_chart_style(fig, ax)` - Застосовує темний стиль до графіків matplotlib.
  - Змінює фон графіка.
  - Змінює колір підписів.
  - Змінює колір осей.

## Функціонал веб-додатку

У веб-додатку доступні такі інтерактивні елементи:

- `Оберіть індекс`
  - Dropdown список для вибору одного з показників:
    - `VCI`
    - `TCI`
    - `VHI`

- `Оберіть область`
  - Dropdown список для вибору області України.

- `Оберіть інтервал тижнів`
  - Slider для вибору діапазону тижнів.

- `Оберіть інтервал років`
  - Slider для вибору діапазону років.

- `Сортувати за зростанням`
  - Checkbox для сортування таблиці за зростанням значень обраного показника.

- `Сортувати за спаданням`
  - Checkbox для сортування таблиці за спаданням значень обраного показника.

- `Скинути фільтри`
  - Button для повернення всіх фільтрів до початкового стану.

### Вкладки результатів
У вкладці `Таблиця` відображаються відфільтровані дані.

Таблиця містить такі стовпці:

- `year` - рік;
- `week` - тиждень;
- `province_name` - назва області;
- обраний показник `VCI`, `TCI` або `VHI`.

### Графік часового ряду

У вкладці `Графік часового ряду` відображається графік значень обраного показника для вибраної області.

Графік будується за:

- вибраною областю;
- вибраним індексом `VCI`, `TCI` або `VHI`;
- вибраним інтервалом років;
- вибраним інтервалом тижнів.

### Порівняння областей

У вкладці `Порівняння областей` відображається стовпчиковий графік середніх значень обраного показника по всіх областях.

Особливості графіка:

- враховується вибраний інтервал років;
- враховується вибраний інтервал тижнів;
- обрана область виділяється окремим кольором;
- інші області відображаються іншим кольором.
